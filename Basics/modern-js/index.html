Modern JavaScript Cheatsheet

Variables and Constants

Use let and const to declare variables.

// Declare a constant
const PI = 3.14;

// Declare a variable
let name = "John";
name = "Doe"; // Reassigning the variable

Arrow Functions

Simplify function expressions with arrow functions.

// Traditional Function
function add(a, b) {
    return a + b;
}

// Arrow Function
const add = (a, b) => a + b;

Template Literals

Embed expressions in strings using backticks (`).

const name = "John";
const greeting = `Hello, ${name}!`;
console.log(greeting); // "Hello, John!"

Array Methods

  • map(): Creates a new array by applying a function to each element.
  • filter(): Filters elements based on a condition.
  • reduce(): Reduces an array to a single value.
// Example: map
const numbers = [1, 2, 3];
const doubled = numbers.map(num => num * 2);

// Example: filter
const even = numbers.filter(num => num % 2 === 0);

// Example: reduce
const sum = numbers.reduce((total, num) => total + num, 0);

Modules

Use ES6 modules for better code organization.

// Exporting a function
export const greet = (name) => `Hello, ${name}!`;

// Importing a function
import { greet } from './greet.js';
console.log(greet('John'));

Promises and Async/Await

Handle asynchronous operations with promises or async/await.

// Using Promises
fetch('https://api.example.com')
    .then(response => response.json())
    .then(data => console.log(data))
    .catch(error => console.error(error));

// Using Async/Await
const fetchData = async () => {
    try {
        const response = await fetch('https://api.example.com');
        const data = await response.json();
        console.log(data);
    } catch (error) {
        console.error(error);
    }
};
