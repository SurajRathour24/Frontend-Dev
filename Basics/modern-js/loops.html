JavaScript Loops - A Complete Guide

1. For Loop

The for loop is used to execute a block of code a specific number of times.

Example:

for (let i = 0; i < 5; i++) { console.log(i); }

2. While Loop

The while loop executes as long as the specified condition evaluates to true.

Example:

let i = 0; while (i < 5) { console.log(i); i++; }

3. Do-While Loop

The do-while loop will execute the code block at least once, then repeat as long as the condition evaluates to true.

Example:

let i = 0; do { console.log(i); i++; } while (i < 5);

4. For...of Loop

The for...of loop iterates over iterable objects, such as arrays or strings.

Example:

const arr = [10, 20, 30]; for (const num of arr) { console.log(num); }
