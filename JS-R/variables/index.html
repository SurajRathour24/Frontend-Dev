Variables and data types in JS

Do not use the VAR in the modern JS
  • Var - is Global scoped
  • Let - Block Scoped
  • Const - Block Scoped

Rules for Variable names

  • Rule 1: Variable name Never start with a Number
  • Rule 2: You can use letter digit underscore dollar.
  • Rule 3: Don't use VAR in modern JS
  • Rule 4: Javascript names are Case Sensitive
