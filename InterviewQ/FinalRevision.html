JavaScript Interview Notes

Q1: What are Variables?
Variables are like containers used to store data values in JavaScript. We can create variales with three types
  • Var - Can redeclear and reinitialize
  • Let - can reinitialize but can't declare ❌
  • Const - can't reinitalize ❌ or redeclear ❌

// Example
var a = 10;
let name = "Suraj";
const pi = 3.14;
Q2: Difference between var, let, and const?
var: function scoped let: block scoped const: block scoped and cannot be reassigned 

// var
var x = 10;
x = 20; // allowed

// let
let y = 30;
y = 40; // allowed

// const
const z = 50;
// z = 60; ❌ error
